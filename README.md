# OWBN Board (Deprecated)

**This plugin is not deployed and is no longer maintained.**

An early modular dashboard framework with Chronicle, Coordinator, and Exec tools as self-contained modules. This approach was superseded by [owbn-chronicle-manager](https://github.com/One-World-By-Night/owbn-chronicle-plugin) (for entity management) and [owbn-client](https://github.com/One-World-By-Night/owbn-client) (for cross-site display and infrastructure).

This repo is kept for historical reference only.
